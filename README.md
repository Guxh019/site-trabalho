# Website Sistema P-MAX

Este é um website estático para apresentar o Sistema P-MAX, um sistema de gestão de ferramentas de corte focado em redução de custos, otimização de produção e controle de estoque. Inclui detalhes sobre benefícios e pré-projeto mecânico completo.

## Estrutura do Projeto

- `index.html`: Página principal com o conteúdo do site.
- `style.css`: Estilos CSS para layout responsivo e design profissional.
- `script.js`: JavaScript para funcionalidades básicas, como rolagem suave nos links de navegação.

## Como Usar

1. Abra o arquivo `index.html` em um navegador web para visualizar o site.
2. Para desenvolvimento local, você pode usar um servidor simples:
   - Com Python: `python -m http.server 8000`
   - Com Node.js: Instale `http-server` e execute `http-server`.
3. Navegue pelas seções usando o menu de navegação.

## Funcionalidades

- Design responsivo para dispositivos móveis e desktop.
- Rolagem suave para seções.
- Layout limpo e profissional com tema azul para negócios.

## Troubleshooting

- **Site não carrega estilos ou scripts**: Verifique se os arquivos `style.css` e `script.js` estão no mesmo diretório que `index.html`.
- **Links de navegação não funcionam**: Certifique-se de que os IDs das seções correspondem aos hrefs nos links (ex: `#reducao-custos`).
- **Problemas de responsividade**: Teste em diferentes tamanhos de tela; o CSS inclui media queries para mobile.

## Dependências

Nenhuma dependência externa é necessária. O site é puramente HTML, CSS e JavaScript.
